## Usage Example

```python
import numpy as np
import pauli_prop_cuda

# Prepare operator and circuit
operator = np.array([[True, False, True, False]], dtype=bool)
coeffs = np.array([1.0])
gates = np.array([[True, False, False, False]], dtype=bool)
qargs = [[0]]
thetas = [np.pi/4]

# Evolve on GPU (10-100x faster than CPU!)
result_op, result_coeffs, error = pauli_prop_cuda.evolve_by_circuit_cuda(
    operator, coeffs, gates, qargs, thetas,
    max_terms=1000, atol=1e-10, frame='s'
)
```
